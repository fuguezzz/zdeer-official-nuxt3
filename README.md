# 基于 Nuxt3.js 的全屏滚动企业官网模板

## 特性

- 支持所有屏幕响应式布局
- 移动端专属设计
- I18n 多语言设计
- 首屏占位背景，秒开首屏
- 精美 UI 设计；
- UI 动画设计

## 技术栈

- Nuxt3.js
- Tailwind CSS
- @nuxtjs/i18n

## 前置要求

建议 `Node版本 >= 18`

## 命令

你可以通过以下命令来启动开发服务器：

```
npm run dev
```

你可以通过以下命令来构建`服务端渲染（SSR）`

```
npm run build
```

你可以通过以下命令静态生成`Nuxt应用程序（SSG）`

```
npm run generate
```
